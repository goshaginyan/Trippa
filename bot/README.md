# Trippa Telegram Bot

Telegram-бот для управления поездками из мессенджера.

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и главное меню |
| `/new` | Создать новую поездку (пошаговый диалог) |
| `/list` | Показать все поездки (предстоящие + архив) |
| `/edit` | Редактировать поездку |
| `/delete` | Удалить поездку |
| `/help` | Справка |
| `/cancel` | Отменить текущую операцию |
| `/test_remind` | Ручная проверка напоминаний |

Команды также доступны через кнопки `ReplyKeyboardMarkup` в нижней панели чата.

## Возможности

- **Создание поездки** — пошаговый диалог: тип, название, города с датами. Инлайн-календарь для выбора дат.
- **Редактирование** — изменение названия, городов, дат. Добавление/удаление городов.
- **Голосовые сообщения** — распознавание речи через OpenAI Whisper, извлечение данных через GPT-4o-mini. Отправьте голосовое — получите готовую поездку.
- **Напоминания** — ежедневно в 22:00 МСК, за 7 и 1 день до поездки. Тестовое напоминание через 60 сек после создания.
- **Статусы поездок** — архив, в пути (сегодня), обратный отсчёт дней.
- **Mini App** — веб-интерфейс внутри Telegram с REST API (CRUD для поездок).

## Архитектура

Один Python-процесс: Telegram-бот (polling) + aiohttp веб-сервер в одном event loop.

| Файл | Назначение |
|------|------------|
| `main.py` | Все хендлеры, ConversationHandler (16 состояний), календарь, напоминания, голос |
| `config.py` | Загрузка `.env`, экспорт `BOT_TOKEN`, `DATA_DIR`, `OPENAI_API_KEY` |
| `storage.py` | JSON-хранилище: один файл `{user_id}.json` на пользователя |
| `voice.py` | OpenAI Whisper (транскрипция) + GPT-4o-mini (парсинг поездки из текста) |
| `datepicker.py` | Инлайн-календарь (`DatePicker`), русские названия месяцев/дней |
| `web.py` | REST API + Telegram Mini App валидация (HMAC-SHA256) |
| `miniapp/index.html` | UI Mini App |

## Формат данных

```json
{
  "id": "hex-timestamp+random",
  "name": "string",
  "type": "vacation|business|weekend|trip|other",
  "cities": [{"name": "string", "dateFrom": "YYYY-MM-DD", "dateTo": "YYYY-MM-DD"}],
  "notifDays": 1
}
```

## Запуск

```bash
pip install -r requirements.txt
export TRIPPA_BOT_TOKEN="your-token"
export OPENAI_API_KEY="your-key"
python main.py
```

Бот читает `.env` файл автоматически через `config.py` (без зависимостей). Переменные можно задать там или через окружение.

## Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `TRIPPA_BOT_TOKEN` | Токен Telegram бота (обязательно) | — |
| `OPENAI_API_KEY` | Ключ OpenAI для голосовых (обязательно для voice) | — |
| `TRIPPA_DATA_DIR` | Папка для хранения данных | `bot/data/` |
| `PORT` / `WEB_PORT` | Порт веб-сервера | `8080` |

## Деплой
