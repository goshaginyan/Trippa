# Trippa Telegram Bot

Telegram-бот для управления поездками из мессенджера.

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/new` | Создать новую поездку (пошагово) |
| `/trips` | Показать все поездки |
| `/delete` | Удалить поездку |
| `/help` | Справка |
| `/cancel` | Отменить создание поездки |

## Запуск

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.

2. Установите зависимости:
   ```bash
   cd bot
   pip install -r requirements.txt
   ```

3. Задайте токен и запустите:
   ```bash
   export TRIPPA_BOT_TOKEN="your-token-here"
   python bot.py
   ```

## Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `TRIPPA_BOT_TOKEN` | Токен Telegram бота (обязательно) | — |
| `TRIPPA_DATA_DIR` | Папка для хранения данных | `bot/data/` |
