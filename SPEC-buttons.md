# Trippa Bot — Спецификация кнопок

## Раскладка главного меню (ReplyKeyboard)

```
[         ➕ Новая поездка          ]
[ 📋 Все поездки ] [ ✏️ Редактировать ]
[ 🗑 Удалить     ] [ ❓ Помощь        ]
```

## Поведение кнопок

### 1. ➕ Новая поездка

Пошаговое создание поездки:

1. Выбор типа (отпуск, командировка, выходные, поездка, другое)
2. Ввод названия
3. Выбор города (из списка или вручную)
4. Выбор дат (заезд, выезд) через календарь
5. Добавить ещё город или сохранить

### 2. 📋 Все поездки

- Показать список всех поездок (предстоящие, затем архив)
- Под каждой поездкой — inline-кнопки: "✏️ Редактировать" и "🗑 Удалить"
- "Редактировать" — показать выбор поля: Название, Даты, Добавить город
- "Удалить" — запросить подтверждение, при "Да" удалить

### 3. ✏️ Редактировать

- Показать список поездок для выбора (inline-кнопки)
- После выбора поездки — показать варианты: Название, Даты, Добавить город
- Название — ввод нового текста
- Даты — выбор города из списка, затем новые даты через календарь
- Добавить город — выбор города + даты

### 4. 🗑 Удалить

- Показать список поездок для выбора (inline-кнопки)
- После выбора — запросить подтверждение ("Да, удалить" / "Нет")
- При подтверждении — удалить поездку

### 5. ❓ Помощь

- Показать справку по всем командам и кнопкам
